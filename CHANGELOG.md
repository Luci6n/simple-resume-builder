# Changelog

## [0.1.0] - 2026-05-04

### Added
- Initial Vite + React + TypeScript project setup.
- Basic main page layout with creator and preview sections.
- Resume creator form for header, summary, experience, education, projects, skills, awards, extracurricular activities, and languages.
- Live resume preview generated from user input.
- Sample resume data and blank resume data.
- Local storage support for keeping resume input after refresh.
- Reverse layout toggle for switching creator and preview positions.
- Light/dark mode toggle.
- PDF print support for the resume preview.
- MIT license.

### Changed
- Improved Creator page styling and reusable form/button styles.
- Structured the preview page into a more ATS-friendly resume layout.
- Updated date fields to support start/end month values and current-work/current-education states.
- Updated README with project details, setup steps, features, and project notes.

### Fixed
- Fixed layout width issues around the creator and preview sections.
- Fixed date formatting in the resume preview.
- Fixed dark mode behavior so it does not automatically follow the system theme.
- Temporarily disabled delete buttons until the remove behavior is ready.
